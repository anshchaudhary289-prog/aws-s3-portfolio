# AWS S3 Static Portfolio Website

A personal portfolio website hosted on AWS S3 using static website hosting.

## What I did
- Created an S3 bucket on AWS
- Enabled static website hosting
- Configured public access policy
- Uploaded HTML/CSS files and deployed

- HTML / CSS
- AWS Static Website Hosting
